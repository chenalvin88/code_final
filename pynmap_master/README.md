# Introduction
pynmap is small python set of useful routines to project N-body snapshots.

## Getting Started

### Prerequisites
Standard python packages are needed: **numpy**, **scipy**, **matplotlib**.

### Installing
Just use the standard setup install.

```
python setup.py install
```
